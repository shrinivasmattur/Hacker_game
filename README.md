# Cyber Escape: The Hacker's Mission

A cyberpunk stealth game built with React Native, Expo, TypeScript, and Expo Router. The player must navigate through a futuristic environment, avoid security systems, and complete missions.

## Features

- 🎮 Cyberpunk-themed home screen
- 🚶 Player movement using on-screen controls
- 🗺️ Game screen with interactive environment
- 🤖 Security drone gameplay (work in progress)
- 💥 Collision detection
- 🔄 Restart functionality
- 📱 Built with React Native and Expo

## Technologies Used

- React Native
- Expo SDK 54
- Expo Router
- TypeScript
- Git & GitHub

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/shrinivasmattur/Hacker_game.git
```

### 2. Go to the project folder

```bash
cd Hacker_game
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the Expo development server

```bash
npx expo start
```

### 5. Run the app

- Install **Expo Go** on your Android phone.
- Scan the QR code shown in the terminal or browser.
- The app will launch on your device.
